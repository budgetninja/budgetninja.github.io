Project Timeline

This is awesome!

  • August 31 2015

    The beginning

    Beginning of the project. Brainstorming project ideas and designing app proposal.

  • September 6, 2015

    App proposal created

    I created the App proposal and started seeking for students who would be interested in this project. Click here to download the pdf.

  • September 11, 2015

    First general group meeting

    First meeting with the team members. I splited the group into Android developer team, IOS developer team and web developer team. Our group name was called "Budget Ninja" and the app name "Fairwell". By next meeting, I would expect to see some basic UI design mock from each team.

  • September 14, 2015

    Budget Ninja Team website created

    I created a GitHub account for the team and made first commit on budgetninja.github.io.

  • September 17, 2015

    Project Pitch created

    Check out this awesome project pitch created by Winnie!

  • September 23, 2015

    Second general group meeting

    In this meeting, we demonstrated the simple design UI on Android phone and on the simulator. By next meeting, we should have some functions done and the login access page done. We would also need to have the Youtube app demo phase 1 uploaded on Youtube.

  • October 15, 2015

    Third general group meeting

    We had the Android, IOS and web app demo ready on Youtube. Check out the Android demo, web demo and IOS demo here!

  • October 25, 2015

    Fairwell Android app design and implementation created

    I created a document about the design and implementation for Fairwell Android. Check out the link here!

  • October 30, 2015

    Fourth general group meeting

    We talked about the following contents: dashboard, friend list, account setting, notification setting, and smart solve. For the smart solve, we aimed to find the best solution to minimize the number of transaction within the party members. We will talk more about it in next meeting.

  • November 10, 2015

    Fifth general group meeting

    In this meeting, we demoed Android, IOS and web application to everyone and we would have our Youtube demo part II ready latest by tomorrow night.

  • End
    of the
    timeline!
